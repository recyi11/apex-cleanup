# ApexCleanup.bat

Download `ApexCleanup.bat`, then double-click it.

Windows will ask for Administrator permission because the tool needs to close stuck Apex Legends, Steam / EA App, and anti-cheat leftover processes.

This branch only contains the standalone BAT build. The script version is on the `main` branch.
