# ApexCleanup.exe

Download `ApexCleanup.exe`, then double-click it.

Windows will ask for Administrator permission because the tool needs to close stuck Apex Legends, Steam / EA App, and anti-cheat leftover processes.

This branch only contains the self-contained EXE build. Source code and scripts are on the `main` branch.
